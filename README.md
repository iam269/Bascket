# 🏀 Bascket - Basketball Club Website Template

## 📝 Description

**Bascket** is a static website template for basketball clubs and sports-related websites. This project is based on the **Neodunk** WordPress theme as a template and inspiration source. It offers an elegant and responsive design, perfect for presenting sports activities, teams, players, and events.

> **Note**: This is a custom static implementation inspired by the Neodunk theme. The original Neodunk is a WordPress theme developed by Awaiken Themes.

## 🔗 Live Demo

Visit the live demo: **[https://iam269.github.io/Bascket/](https://iam269.github.io/Bascket/)**

## ✨ Main Features

- 🖥️ **Responsive Design**: Perfect appearance on all devices - mobile, tablet, and desktop
- 🎨 **Flexible Customization**: Compatible with Elementor Page Builder for drag-and-drop editing
- 📄 **Pre-defined Pages**: Homepage, about club, teams, players, competition calendar, photo galleries, blog, contact
- 🛒 **WooCommerce Integration**: Full functionality for online sports equipment store
- 📧 **Contact Forms**: Compatible with Contact Form 7
- 🔍 **SEO Optimized**: Structure optimized for search engines
- 🌍 **Translated and Localized**: Full support for Romanian and other languages

## 💻 System Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- Elementor Page Builder (recommended)
- Apache or Nginx web server

## 🚀 Installation

1. 📥 Download the theme file (.zip)
2. 🔐 Access the WordPress admin dashboard
3. 🎯 Navigate to **Appearance > Themes > Add New > Upload Theme**
4. ⬆️ Upload and activate the theme
5. 🔧 Install recommended plugins (Elementor, ElementsKit, Contact Form 7)
6. 📦 Import demo content (optional)
